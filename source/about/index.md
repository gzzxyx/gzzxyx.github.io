# 摄影爱好者协会
{% raw %}<div class="notification is-info">{% endraw %}
广东高州中学摄影爱好者协会成立于2017年9月1日，旨在组织联系一批摄影爱好者，学习交流摄影心得，提高摄影技能，陶冶情操，活跃校园的艺术气氛。影协设立了制作部、理事部、宣传部、技术部，担任日常各部门活动的摄影任务，举办摄影展等。
